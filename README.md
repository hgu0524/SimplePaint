# (C# 코딩) SimplePaint

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 마우스를 이용해 도형을 그릴 수 있는 간단한 그림판 프로그램

- 사용한 플랫폼:
- C#, .NET Windows Forms, Visual Studio, GitHub

- 사용한 컨트롤:
- PictureBox, Button, ComboBox, TrackBar, Label, GroupBox

- 사용한 기술과 구현한 기능:
- Windows Forms를 이용한 GUI 구성
- Bitmap과 Graphics 객체를 이용한 그림 그리기
- 마우스 이벤트(MouseDown, MouseMove, MouseUp)를 이용한 드래그 기능 구현
- ComboBox를 이용한 색상 선택 기능 구현
- TrackBar를 이용한 선 두께 조절 기능 구현
- Button을 이용한 도형 선택 기능 구현
