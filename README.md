# (C# 코딩) SimplePaint

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 기본적인 도형을 선택하고 설정할 수 있는 그림판 앱의 UI 구현

- 사용한 플랫폼:
- C#, .NET Windows Forms, Visual Studio, GitHub

- 사용한 컨트롤:
- PictureBox, Button, ComboBox, TrackBar, Label, GroupBox

- 사용한 기술과 구현한 기능:
- Windows Forms를 이용한 GUI 구성
- ComboBox를 이용한 색상 선택 기능 구현
- TrackBar를 이용한 선 두께 조절 기능 구현
- Button 클릭 이벤트를 이용한 도형 선택 기능 구현
